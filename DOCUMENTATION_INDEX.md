## 🔍 Quick Reference

### How to Use Translations
```tsx
// Method 1: Hook
const title = useTranslatedContent("Welcome");

// Method 2: Component
<TranslatedHeading level={1}>Welcome</TranslatedHeading>

// Method 3: Multiple
const content = useTranslatedContents({ title: "Welcome" });
```
